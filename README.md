# NCM-Chess
This is a chess game coded in python that uses pygame and stockfish to create moves from Next Chess move.

## Used modules
- Pygame: Used for GUI
- chess: Used for chess moves, board setup, chess pieces.
- stockfish: Used for NCM Inegeration.
- os: Used for files path
- random: Used for move delay
- time: Used for delay
- sys: Used for fonts

## Python Deatils
- Version 3.14.0
- Devloped using VS code

## Resources
Resources are stored inside assets. These include textures and audio files.
There is also an exe file converted using pyinstaller inside dist. Unfortunately,
You cannot apply custom textures to the exe file without reconverting the file again.
